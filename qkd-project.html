Quantum Secure Banking (QKD)

A Python-based simulation of a quantum-proof financial system.

The Project Story

Traditional encryption is vulnerable to emerging quantum computers. My solution was to develop a simulation that uses the BB84 Quantum Key Distribution (QKD) protocol to create a provably secure communication channel. The system can detect any eavesdropping attempts in real-time by measuring the Quantum Bit Error Rate (QBER).

The main challenge was bridging theoretical quantum physics with practical Python code using Qiskit. This project provided a deep appreciation for how physics-based principles can create next-generation security solutions.

Technology Stack

  • Python
  • Flask
  • Qiskit
  • MySQL
  • AES Encryption
