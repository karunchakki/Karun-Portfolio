Air Quality Prediction Model

Using Machine Learning to forecast air pollution levels.

The Problem

Air pollution is a major environmental and public health concern. The ability to accurately predict air quality in advance can help authorities issue warnings and citizens take precautionary measures.

My Solution

I developed a data-driven model to predict the Air Quality Index (AQI) using historical data on pollutants (like PM2.5, SO2, NO2) and meteorological factors. The project involved extensive data cleaning, exploratory data analysis to find correlations, and feature engineering to create more predictive inputs. I trained a Decision Tree Regressor model, which proved effective in capturing the non-linear relationships in the data. The model was rigorously validated using cross-validation techniques to ensure its reliability.

Challenges & Lessons Learned

Dealing with missing data points and outliers in the historical dataset was a significant challenge that required careful imputation strategies. This project was a deep dive into the practical application of the machine learning workflow, from data sourcing and wrangling to model training, evaluation, and interpretation.
