Smart Pulse Oximeter

An IoT device for real-time remote health monitoring.

The Project Story

Continuous monitoring of vital signs is crucial for managing chronic health conditions, but traditional methods are often restrictive or intermittent. To solve this, I engineered a wearable Smart Pulse Oximeter using an ESP32 microcontroller that provides a continuous data stream for analysis.

The device integrates a MAX30102 sensor to measure heart rate and blood oxygen saturation (SpO2), along with a DS18B20 sensor for body temperature. Data is displayed on a local OLED screen and transmitted to a cloud platform for remote monitoring. The biggest challenge was accurately calibrating the bio-sensors and implementing algorithms to filter out motion artifacts. This project was an excellent fusion of my skills in biomedical instrumentation, embedded systems, and cloud integration.

Technology Stack

  • ESP32 (C/C++)
  • MAX30102 Sensor
  • DS18B20 Sensor
  • IoT Cloud Platform
